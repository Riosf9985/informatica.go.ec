HTML

1¿Que es HTML?

HTML, siglas en inglés de HyperText Markup Language, hace referencia al lenguaje de marcado para la elaboración de páginas web.Ademas es el lenguaje con el que se define el contenido de las páginas web. Básicamente se trata de un conjunto de etiquetas que sirven para definir el texto y otros elementos que compondrán una página web, como imágenes, listas, vídeos, etc.

2Ventejas Y Desventajas

Sus Ventajas serian Que es Facil de Usar y Ademas Te ayuda a Entender Mejor El uso de las etiquetas basicas, haciendola el primer escalon para La creacion de Paginas Web. Ademas de que no se nesecita instalar, ya que viene por defecto En Tu PC O laptop.
Las Desventajas es que Hay Hay otros programas que son mas eficientes Como Sublime Text o Visual Studio Code, que aunque nesecitan instalarse son mas rapidos y eficientes de usar que HTML.

Estructura Basica en HTML

1¡EMPECEMOS!

La extructura Basica en HTML se comforma por las etiquetas 'html, head, title y el body'.

El html Sirve para mostrar desde donde comienza nuestra pagina.
El Head define la cabeza de nuestra pagina. Y dentro de eta etiqueta tambien usares mos el title el cual nos ayudara a ponerle un titulo a la Pagina Web.
Y el Body nos ayudara a definir el Cuerpo de Nuestra Pagina.
Recuerda que Todas estas etiquetas se cierran agregandole un "/" a la etiqueta de esta forma: < title > y < /title > .

Etiquetas de Parrafo

1Etiquetas de Parrafo

Estas ayudan a modificar el estilo de nuestro parrafo como:

Enlaces en HTML

1¿Que son los enlaces?

Los enlaces son posiblemente la Cosa mas importante de HTML, ya que conesta podemos comunicar una pagina web con otra. para crear un enlace se usa la etiqueta <a> con la siguiente sintaxis: <a href="enlace.html" target="_blank" title="Titulo de mi destino de enlace "> Ir a otra pagina </a> .

2Atributos del <a>

Los atributos del <a> son:

href= :Nos permite Ingresar un URI hacia otra Pagina Web.
target= :Nos permite darle un destino a nuestro enlace. Este atributo tambien tiene otros atributos como: '_blank','_self','_parents','_top'. Pero no explicare su funcionamiento en esta pagina debido a que eso es tema para otra pagina.
title= :Permite ponerle un titulo a nuestro enlace.
name= :Nos ayuda a marcar una parte del Documento, ya que si creamos un enlace con este atributo, este se definira como posicion y no como enlace de navegacion.

Insertar Audio

1La etiqueta <audio>

La etiqueta <audio> sirve para insertar un archivo de audio que tengamos en nuestra computadora y ponerla en nuestra pagina web. Y su estructura es la siguiente: <audio> src="musica.mp3" autoplay, controls, loop, muted, preload </audio>

2Atributos que aconpañan al <audio>

Los atributos que acompañan al <audio> son:

autoplay :Indica que el audio se reproducira solo al estar listo.
controls :Permite mostrar los controles del audio.
loop :Indica que el audio se reproducira de nuevo cada vez que termine.
muted :Indica que el audio estara silenciado.
preload :Indica si el audio carga previamente al tiempo que la pagina a cargado. Sus valores son Auto o Home.
src= :Indica la ubicacion del archivo de audio.
<source> :Sirve para añadir mas de una fuente de audio.

Las Anclas

1¿Que es un Ancla?

Un Ancla es una etiqueta que esta hecha para enviarnos a alguna parte de nuestra pagina u a otra pagina de forma mas rapida y facil

La extructura de un ancla en HTML es la siguiente:
<a href="#Las Nombre que le pondremos al Ancla">Nombre con el que aparecera en la Pagina web</a>
. Esta la pondremos cerca del inicio de la pagina ya que esta nos permite poner la parte principal del Ancla por haci decirlo.
<a name="Nombre que le pusimos al Ancla">
. Y esta la pondremos en la parte de la pagina en la cual queremos que el ancla nos lleve.

Caracteres Especiales

1¿Que son los Caracteres Especiales?

Los caracteres Especiales son cualquier caracter que requieran de un Codigo ASCII para poder mostrarse en una pagina web. Estos se muestran encerrando en codigo ASCCI del caracter que queremos mostrar entre &# y ;. Por ejemplo, El Codigo ASCII del @ es el 64 y e representa asi: &#64;.

2 Algunos Caracteres Especiales

El Formato de Listas

1¿Que son las Listas en HTML?

Las listas en HTML son Etiquetas que nos permiten Realizar listas en las paginas web de una menera simple y efectiva Estas listas nos sirven para Realizar diferentes tipos de listas con las etiquetas ul, ol, dl y el li.

2 Tipos de Listas

Las listas se clasifican Listas Ordenadas, Desordenadas y de Definicion. En las listas Ordenadas se usa la etiqueta<ol> que signidica "Orderest List " . En esta podemos presentar listas con esquemas numerados los cuales van en Orden. Extructura = <ol><li> Primer Elemento</li>
<li>Segundo Elemento </li></ol>.
Luego tenemos la Lista desordenada en la que se usa el <ul> la cual significa "Unorderest List" . En esta podemos presentar listas con viñetas. Extructura = <ul type="disc"><li> Primer Elemento</li> <li>Segundo Elemento </li></ul>.
Y por Ultimo tenemos las Listas de Definición. En Estas No se usan el ul, ol ni el li, sino que aqui se usan las etiquetas <dl> <dt> y <dd>. Extructura = <dl> <dt>Texto</dt></dd>Texto.</dd><dt>Más Texto</dt><dd>Aun Más Texto.</dd></dl>
